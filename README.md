# Medieval Platformer Game

A simple medieval-themed platformer built with Phaser.js. Battle enemies, jump across platforms, and progress through multiple levels.

## How to Play
- Open `index.html` in Chrome.
- Use arrow keys to move and jump.
- Space: Swing sword
- Shift: Shoot arrow

## Features
- Multiple levels
- Enemies to defeat
- Sword and bow/arrow attacks
- Player wears armor and helmet
